## 🚀 Overview
This is a second-generation automation pipeline designed for the **Ashley MindShift** brand. It features a unique "Dark & Deep" aesthetic with cyan-glow subtitles and a slower, more emotional storytelling pace.

## 📅 Features
- **5x Daily Posting**: Automated schedule optimized for USA peak hours (9am, 12pm, 3pm, 6pm, 9pm EST).
- **100+ Viral Topics**: Pre-loaded with high-engagement psychological triggers and behavioral analysis.
- **Auto-Cleanup**: Automatically deletes Supabase log entries older than 3 days to keep the project on the free plan limits.

## 🛠️ Project Structure
- `core/ai_script.py`: The "Brain" - generates scripts with deep solutions and scarcity triggers.
- `core/tts.py`: The "Voice" - ChristopherNeural at a slower, tensed rate.
- `core/yt_scraper.py`: The "Eyes" - fetches cinematic, text-free dark B-roll.
- `core/video_editor.py`: The "Hands" - assembles clips with neon cyan styling.
- `core/supabase_db.py`: The "Memory" - handles logging and auto-cleanup.

## 📦 Setup & Deployment
1. **Local Install**: `pip install -r requirements.txt`
2. **Environment**: Rename `.env.example` to `.env` and add your API keys.
3. **Authentication**: Run `python refresh_token.py` to link your YouTube channel.
4. **GitHub Cloud**: 
   - Push to your repo: `https://github.com/sibby-killer/yt_lina_auto.git`
   - Add your `.env` keys as **GitHub Secrets**.
   - Enable **GitHub Actions** to start the 5x daily schedule.
