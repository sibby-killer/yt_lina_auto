## Script Architecture — 5-Part Framework

Every script MUST follow this exact structure:

### PART 1 — HOOK (3-5 seconds)
The first line must **stop the scroll**. Rotate between these hook types:

| Hook Type | Example |
|-----------|---------|
| Scenario | "Imagine someone is trying to manipulate you..." |
| Question | "Have you ever noticed why some people always get what they want?" |
| Bold Statement | "The more you chase someone the more they run away from you." |
| Curiosity | "There is a secret technique that psychology has hidden from you." |
| Challenge | "99 percent of people fall for this manipulation trick every single day." |

### PART 2 — PSYCHOLOGY LABEL (3-5 seconds)
Name a specific psychology concept to build credibility.
- Makes viewer feel they are learning something scientific and rare
- Example: "In psychology this is called the Scarcity Effect"

### PART 3 — MAIN EXPLANATION (25-35 seconds)
Break down the concept using:
- Vivid relatable scenarios
- Contrast technique (first THIS then THIS)
- Power phrases ("here is the secret", "now the real game begins")
- Short punchy sentences
- Direct YOU addressing
- If numbered format: exactly 3 points

### PART 4 — CONSEQUENCE / PAYOFF (5-8 seconds)
Show what happens when they apply or ignore this knowledge.
- Create empowerment feeling
- Viewer should feel they now hold secret power

### PART 5 — CTA WITH URGENCY (3-5 seconds)
End with urgency/scarcity based call to action.
- "We might never meet again so subscribe to Ashley MindShift right now"
- "Before this video disappears subscribe and stay connected"
- Never use plain "subscribe" — always wrap in urgency

## Language Rules

### DO:
- Always use YOU — speak directly to viewer
- Use short punchy sentences
- Sound confident — never use maybe or I think
- Use conversational English
- Use power phrases
- Use rhetorical questions
- Paint vivid mental pictures

### DO NOT:
- Use complex academic language
- Sound like a textbook
- Write longer than 60 seconds
- Use emojis or hashtags in script body
- Add stage directions or editing notes
- Repeat same hook type consecutively
- Repeat same CTA consecutively

---

## Quality Checklist

- [ ] Hook grabs attention in first 3 seconds
- [ ] Contains exactly ONE psychology concept
- [ ] Uses direct YOU addressing throughout
- [ ] Tone is conversational not academic
- [ ] Has emotional triggers (minimum 2)
- [ ] Contains power phrases
- [ ] Ends with urgency-based CTA
- [ ] Read time is 40-60 seconds
- [ ] Word count is 100-160 words
- [ ] Feels like hidden knowledge being revealed
- [ ] Different hook type from previous script
- [ ] Different CTA from previous script
- [ ] Viewer feels empowered after hearing it
